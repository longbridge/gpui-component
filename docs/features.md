# 功能

## 多数据库管理

OnetCli 把多种数据库连接放进同一个桌面界面中，适合日常查询、结构浏览和数据处理。

当前仓库文档中明确提到的数据库能力包括 PostgreSQL、MySQL、SQLite、SQL Server、Oracle、ClickHouse、Redis 和 MongoDB。

## SSH 与 SFTP

OnetCli 支持 SSH 远程终端与 SFTP 文件管理，让数据库排查、服务器维护和文件传输可以沿用同一套连接上下文。

## 本地终端

OnetCli 内置本地终端，适合执行脚本、查看输出和处理与数据库、部署相关的命令行任务。

## AI 助手

OnetCli 内置 AI 助手，可用于自然语言生成 SQL、解释查询以及辅助数据分析。
