# 文档

OnetCli 当前官网先保持最小集合，这一页负责把常用入口收拢到一起。

## 快速开始

1. 先从 [GitHub Releases](https://github.com/feigeCode/onetcli/releases) 下载对应平台版本。
2. 启动应用后，按你的工作流添加数据库连接、SSH 主机或本地终端标签页。
3. 如果你需要 AI 辅助分析，可以在应用内继续使用查询解释、自然语言生成 SQL 等能力。

## 核心入口

- [功能总览](/features)
- [下载页面](/download)
- [更新日志](/changelog)
- [GitHub 仓库](https://github.com/feigeCode/onetcli)

## 适合谁使用

- 需要同时处理数据库、服务器与终端任务的开发者
- 需要把 SSH、SFTP 和数据库排查放到同一工作台的运维同学
- 希望在桌面端直接结合 AI 做 SQL 和数据分析的工程团队
