# Docs I18n Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Add a Simplified Chinese docs tree under `/zh/docs/` with a built-in locale switch, while keeping home and shared pages reusable.

**Architecture:** Configure VitePress with `root` and `zh` locales, generate separate sidebars from mirrored English and Chinese docs directories, and add lightweight `/zh/*` wrapper pages for shared content. Keep shared Vue pages single-source and compute locale-sensitive links from the current route.

**Tech Stack:** VitePress 2, Vue 3, Bun, vitepress-sidebar

---

### Task 1: Locale-aware site configuration

**Files:**
- Modify: `docs/.vitepress/config.mts`

**Step 1: Define separate English and Chinese sidebar generators**

Create one sidebar for `/docs/` and one for `/zh/docs/`.

**Step 2: Add `locales` entries for `root` and `zh`**

Set locale labels, language codes, locale-specific navigation, footer text, edit links, and sidebars.

**Step 3: Keep shared top-level site options unchanged**

Preserve `base`, markdown theme setup, and existing plugin configuration.

### Task 2: Shared page locale routing

**Files:**
- Modify: `docs/index.vue`

**Step 1: Read the active route locale**

Use the current route path to detect whether the page is under `/zh/`.

**Step 2: Compute localized links**

Send the primary CTA and components link to either `/docs/...` or `/zh/docs/...`.

### Task 3: Chinese route wrappers

**Files:**
- Create: `docs/zh/index.md`
- Create: `docs/zh/contributors.md`
- Create: `docs/zh/skills.md`

**Step 1: Reuse existing Vue components**

Create thin Markdown wrapper pages that import the shared Vue files.

**Step 2: Add Chinese page metadata**

Set Chinese titles and descriptions where helpful.

### Task 4: Chinese docs tree scaffold

**Files:**
- Create: `docs/zh/docs/**`

**Step 1: Mirror the English docs path structure**

Create a Chinese file for every English file under `docs/docs`.

**Step 2: Provide translation placeholders**

Add Chinese headings and a short note when a page has not yet been translated.

### Task 5: Verification

**Files:**
- Verify build output only

**Step 1: Install docs dependencies if needed**

Run: `bun install`

**Step 2: Build the docs site**

Run: `bun run build`
Expected: build succeeds and locale routes are generated.

**Step 3: Review changed files**

Run: `git diff -- docs/.vitepress/config.mts docs/index.vue docs/zh`
Expected: only locale-related docs files are changed.
